## 证据强度说明

与 [guides/report_principles.md](../guides/report_principles.md) §1.1 一致。

| 强度 | 标准 |
|------|------|
| 强 | 多源交叉,≥3 个独立来源**且来源类型不同**(如访谈 + 问卷 + 行为数据 / 第三方报告)。一手行为证据 ≥ 口头偏好。 |
| 中 | 同类多源,≥3 条独立访谈一致;或单源但细节丰富、有理论支撑。 |
| 弱 | 单条访谈 / 单次观察 / 二手转述 / 间接推断。结论应降级为假设或加限定条件。 |
| 待验证 | 有信号但证据不足。需在"待解问题"章节列出,建议补充研究方法。 |

**判定提示**:

- "强" 的门槛是**类型多样性**,不只是数量。10 条访谈 = 中,不是强;3 条访谈 + 1 份问卷 + 1 份行为数据 = 强。
- 行为证据(用户做了什么)优先级高于口头偏好(用户说会做什么)。
- 单条访谈即使再有力,也只能定为"弱"(假设级),不能支撑战略级判断。
